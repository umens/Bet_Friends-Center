src/app/core/http/http.service.ts

HTTP_DYNAMIC_INTERCEPTORS
HTTP_DYNAMIC_INTERCEPTORS:
Default value : new InjectionToken<HttpInterceptor>('HTTP_DYNAMIC_INTERCEPTORS')

Allows to override default dynamic interceptors that can be disabled with the HttpService extension. Except for very specific needs, you should better configure these interceptors directly in the constructor below for better readability.

For static interceptors that should always be enabled (like ApiPrefixInterceptor), use the standard HTTP_INTERCEPTORS token.
